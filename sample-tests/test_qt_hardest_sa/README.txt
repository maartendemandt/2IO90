Python 3 is required, preferably with python in system PATH

INSTRUCTIONS
1. Clear all previous test samples and results with clear_io.py
2. Run generate_all.py to generate all test samples. These will be stored in the input folder.
3. Run test_all.py to test the whole batch of test samples. The output files while stored in the output folder.

Edit samples.csv to change the test sample parameters and see the results.
